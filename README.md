# Neon Bricks
Drop colorful blocks, chain combos, and activate game-changing power-ups like **magnetic attraction** and **slow-motion** to build the ultimate tower.

## Features
- Satisfying stacking with particle explosions
- Multiple power-ups (slow-mo, magnetic, multipliers)
- Combo system
- Increasing difficulty
- Clean neon/2030 visual style

## How to Play
- **SPACE** — Drop brick
- **P** or **1-7** — Activate power-ups
- **ESC** — Pause
