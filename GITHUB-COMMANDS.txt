# 🚀 GITHUB + VERCEL DEPLOYMENT COMMANDS
# Cross-platform ready for Desktop, iPhone, and Android!

# STEP 1: CREATE GITHUB REPOSITORY
# 1. Go to github.com
# 2. Click "New repository" 
# 3. Name: "neon-bricks"
# 4. Make it PUBLIC (for Vercel free tier)
# 5. Don't add README, .gitignore, or license
# 6. Click "Create repository"

# STEP 2: PUSH TO GITHUB
# Your GitHub username: stampedit
# Repository name: neon-bricks

git remote add origin https://github.com/stampedit/neon-bricks.git
git branch -M main
git push -u origin main

# STEP 3: DEPLOY TO VERCEL
# 1. Go to vercel.com
# 2. Sign in with your GitHub account
# 3. Click "New Project"
# 4. Import your "neon-bricks" repository
# 5. Vercel will auto-detect it's a static site
# 6. Click "Deploy"

# 🎮 YOUR GAME WILL BE LIVE AT:
# https://neon-bricks.vercel.app
