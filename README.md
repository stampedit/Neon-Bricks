# 🎮 Neon Bricks - Enhanced Stacking Game

![Neon Bricks](https://img.shields.io/badge/Platform-Web-blue.svg)
![License](https://img.shields.io/badge/License-MIT-green.svg)
![Status](https://img.shields.io/badge/Status-Active-brightgreen.svg)

> **A futuristic, addictive stacking game with power-ups, combos, and cross-platform support.** Stack blocks perfectly, collect power-ups, and build the highest score!

## 🌟 Features

### 🎮 Core Gameplay
- **Addictive stacking mechanics** - Time your drops perfectly!
- **Power-up collection system** - Stack special blocks to gain abilities
- **Combo system** - Build chains for massive scores
- **Progressive difficulty** - Speed increases as you play
- **High score tracking** - Beat your personal best

### ⚡ Power-Ups
- **🕐 Time Dilation** - Slows down block movement
- **🧲 Gravity Well** - Auto-aligns blocks perfectly
- **🛡️ Quantum Shield** - Protects from one mistake
- **x2 Score Amplifier** - Double your points
- **❄️ Freeze** - Stops block movement temporarily

### 📱 Cross-Platform
- **Desktop**: Keyboard (SPACE), Mouse Click, ESC for menu
- **Mobile**: Touch controls, on-screen buttons
- **Responsive Design**: Works on all screen sizes
- **PWA Ready**: Install as native app on phones

### 🎨 Visual Features
- **Neon aesthetic** with glowing effects
- **Particle systems** for visual feedback
- **Smooth animations** and transitions
- **Mobile-optimized UI**

## 🚀 Quick Start

### Play Online
🌐 **Live Game:** [https://neon-bricks.vercel.app](https://neon-bricks.vercel.app)

### Local Development
```bash
# Clone the repository
git clone https://github.com/stampedit/neon-bricks.git
cd neon-bricks

# Start a local server
python -m http.server 8000
# OR
npx serve .

# Open in browser
http://localhost:8000
```

## 🎮 How to Play

### Desktop Controls
- **SPACE** - Drop current block
- **Mouse Click** - Drop block
- **P** - Use next power-up
- **1-7** - Use specific power-up
- **ESC** - Pause/Menu

### Mobile Controls
- **Tap anywhere** - Drop block
- **⚡ Power-up Button** - Use next power-up
- **🔄 Restart Button** - Quick restart
- **🏠 Home Button** - Back to menu
- **Numbered slots** - Use specific power-ups

### Gameplay Tips
1. **Stack blocks perfectly** for combo bonuses
2. **Collect power-ups** by stacking special blocks
3. **Use power-ups strategically** - save them for difficult moments
4. **Build combos** for massive score multipliers
5. **Watch the speed** - it increases as you progress!

## 🎯 Game Mechanics

### Scoring System
- **Base points**: 50 per successful stack
- **Perfect placement**: 2x points
- **Combo multiplier**: Increases with consecutive perfect stacks
- **Power-up multiplier**: Some power-ups double points

### Difficulty Progression
- **Speed increases** every 10-15 seconds
- **Level increases** every 10 stacks
- **Landing zones** narrow over time
- **Block patterns** become more challenging

### Power-Up System
- **20% chance** for power-up blocks
- **Inventory system** - Collect up to 7 power-ups
- **Strategic usage** - Save for critical moments
- **Visual indicators** - Icons and colors for each type
